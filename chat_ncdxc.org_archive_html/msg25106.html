Getting a confirmation of receipt for the report given (other than some contests which have their own set of rules) is not being moralistic by any stretch.

It is PART of the contact; signifying an ending, a successful exchange COMPLETED.  Each mode has it's own format.  Without this closure, contact validity is in question.

Accepting less than that, is where integrity enters; and THAT is a personal choice.  The log entry is either confirmed or is a hopeful attempt.  Either LOTW (Clublog, others) or the contest judges sort that out later.

I (like most) draw the line at one 73 reception, requiring more is simply a waste of time better spent working the next contact.  If the other side doesn't get my 73 and sends a report again, it is my option to repeat it for them (I usually do that in this case).  But by then it's logged on my side, so if they didn't get my 73 ever, they'll still be able to confirm later.

Even the JT software accepts that as a standard (except in fox/hound mode, a different set of rules on the fox/DX side).

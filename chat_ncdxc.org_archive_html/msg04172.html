> Has any Mac user have any suggestion about a very fine, easy to learn & use, logging program for a Mac?

> kind regards....harry (KX6C)

Without question, the best logger on the Mac is MacloggerDX: http://dogparksoftware.com/MacLoggerDX.html

The feature list goes on and on: logging, rig control, QRZ integration, cluster integration, awards tracking, etc. You can use the basic features at first, and learn/add complexity as you wish (or not). I have been using it for years, and am probably one of the heaviest "power users" around. Don Agro provides amazing tech support, often fixing problems (and adding features) in *minutes* after an email.

It integrates tightly with Clublog, such that you can upload QSOs in real-time on a QSO-by-QSO basis. For LotW, there are scripts to semi-automatically export log entries, digitally sign them, and upload. MacloggerDX also takes the LotW "QSO Report" and uses it to mark log entries as "Confirmed" where appropriate.

You can try the 15-minute (fully featured) demo version, or you are welcome to come by my shack to see everything the program does. There is hardly a feature that I don't use.

One caveat: If you want to do *serious* contesting, this is NOT the program for you. It is just fine for casual contesting (there is support for contest exchanges, etc.), but it's not a replacement for N1MM or similar contest loggers.

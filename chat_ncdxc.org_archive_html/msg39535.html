On 5/6/2025 9:46 AM, Paul N6PSE via Chat wrote:
The Clublog Livestream feature requires a very robust internet connection between the DX Station and the Clublog servers in the EU. Livestream was created to reduce dupe contacts but with its inherent problems, it actually increases the number of duplicates. 

If the connection falters in any way, the Livestream feature will fail to show the current status of the DX Station and the stations being worked. 

If you get an RR73 from the DX Station, you should be satisfied that you are in the log. 

Paul N6PSE

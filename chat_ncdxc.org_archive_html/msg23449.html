Hi Bob,
While I agree with Ed on DX Labs, it can be a little intense to get acclimated with in the beginning.
Although I use DX Lab for chasing DX, my main go-to for easy day-to-day logging is N3FJP. It has all the items you mentioned and more. 
Once you purchase the N3FJP software set, you'll receive lifetime free upgrades / updates to any new application.
I find it very easy to use as casual daily logger. It also has applications that work well as a contest logger too.

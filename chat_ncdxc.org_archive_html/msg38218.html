FOR SALE: FlexRadio 6600 signature series with internal GPSDO module option. $2900

 

I am parting with this unit as I now also own the FlexRadio 8600 and need to make room. This is now my 3rd FlexRadio. This unit was purchased June 20th, 2023, therefore ONLY 1 ½ years old. New price at FlexRadio is $4599 for the 6600 and $3499 for the used ones if purchased from them directly. The GPSDO was an extra $650 option, was factory installed. This radio was over $5K new with the features before tax.

 

For pictures see: https://www.flexradio.com/products/certified-pre-loved-flex-6600-signature-series-sdr-transceiver/

Identical to this unit, except it has the actual SMA gold connectors for GPS and 10MHz reference since it also has the GPSDO.

 

This is the non-M version, meaning NO knobs just a power button, fully controlled by SmartSDR. On version 3.8.2, latest firmware. Works fine on W10 and W11 or the Mac but you need to purchase software for the Mac. Windows software is licensed via the hardware and this is on latest version.

 

This is the unit with the two full SDR receivers with their own antennas, but supports up to 4 slices (receivers) using 2 antennas at the same times and all can be on different bands. There are also 2x RX only ports and 2 ports which can be used for transverters. Also supports diversity reception. The 6400 does not do all this. I can work on 20m and 6m at the same time during the day and 40m and 160m at night for example, and not miss anything.

 

Having the optional GPSDO creates rock solid frequency stability and accuracy. No drift and exact frequency at all times. Even more important if used with a transverter. I had this working with the Q5 Signal 5 band version transverter flawlessly giving me everything up to 1.2GHz, for the most part. (Transverter not included, I will be using it with the 8600)

 

Includes:

  • 6600 FlexRadio with GPSDO (installed)
  • GPS indoor antenna for GPSDO (New in wrapper) I used an outdoor antenna
  • FHM-3 Microphone (new in box) I used RadioSport with it so never opened it
  • Anderson power pole cable (new)
  • Ethernet cable (new)
  • Quick start guide
  • Original shipping box/foam/wrap, etc.

 

Pickup is OK or we can arrange something locally. I can also ship it in the original box, double boxed via UPS. Shipping cost we can figure out if applicable.

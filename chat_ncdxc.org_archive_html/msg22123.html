Al, Sadly, there is so little CW activity these days. I go where the activity is and that is predominately on FT8/FT4.  Even many of the stalwart CW guys are now doing FT8. 

Last night, one could just about get DXCC on 80 meters using FT8. There were so many EU stations that were relatively easy

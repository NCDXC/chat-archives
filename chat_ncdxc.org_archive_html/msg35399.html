Set correctly. most radios, and probably all modern radios not stuck with a single 2.4-2.7 KHz filter do just fine with a broad passband, with use of an AM or FM filter. That brings up a question I’ve had since first operating FT8. In the initial publication introducing F/H mode in May, 2018, Joe Taylor wrote “Hounds make initial calls anywhere in the range 1000-4000 Hz”. Almost all of all have radios that can be set for a broader passband than we are currently using. That would open up an additional full KHz for calling. Does anyone have any idea why this never happened?

73, Bruce K3NQ

He is indeed having issues with completing QSOs. As of this afternoon (Monday, 27 May) he's changed things up a bit after going to 10m. He's reduced things to one or two streams, and has moved his transmit frequency up to the middle of the passband on 28091. Some computers have problems below 500 Hz, so he's trying this to see if it helps at all. So if anybody was doubting that with this change in operating pattern of it was a legit signal, the chief pilot confirmed it's really him.

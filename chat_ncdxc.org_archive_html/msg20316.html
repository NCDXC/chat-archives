On Jun 22, 2020, at 4:19 PM, Christopher Hoover via Chat <chat@ncdxc.org> wrote:

I will repeat my suggestion: FT8 spots should only include the call and the band.   Add the dial frequency for F/H or for bands where there are multiple dial frequencies (e.g. 6m).

There's no point to give even/odd or the passband frequency.  It is not helpful information, but it can be abused.

If the software can't decode the DX, no call should be made.

This is the same as CW or PHONE, except for the use of a wetware decoder.   Don't call until you can decode the DX.

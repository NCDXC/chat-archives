Most EU DXpeditions favor MSHV. MSHV is a Bulgarian derivative that emulates most features of WSJT-X. MSHV DX can be Odd or Even, whereas a WSJT-X Fox will always be Even.   When you see a Hound moved down to the DX station's tones, it merely indicates that the Caller was a Hound, nothing more. It does not indicate if the DX station was MSHV or a Fox. 

Recently, I have seen many MSHV stations, including T22T having significant issues in "hand shaking" with their callers. It is because this "bastardized" software is not native and Co-developed with the WSJT-X Team. Just as JTDX has its own issues for the same reasons. MSHV has some secret features built into it that make it act as a robot. 

IF you want the best FT4/8 mode experience, use the latest release for general use WSJT-X. If you are not sure that the DX is a Fox or MSHV, you can call as a Hound as long as the DX is Even. 

Good luck,

Paul N6PSE

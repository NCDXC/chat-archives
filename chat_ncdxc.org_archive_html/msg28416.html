This is F/H mode.    You cam tell because there are multiple streams (messages) at the same passband frequency (611 Hz).

MSHV aka multi stream sends multiple streams, but at close but different passband frequencies.

I get multiple callers at the same time and wonder how to work them all together instead of one by one.

F/H mode as a fox is meant for a "special operating activity", i.e., rare entity and/or high sustained Q rates (> 100/hr).  

(Does VE2CSI qualify?   Dunno.  I notice special event stations are doing this more often.)

It would be unfriendly to run it as a regular  station on the normal FT8 sub-bands, not to mention confusing.

Same is said of running MSHV on the normal FT8 sub-bands.

On Wed,1/21/2015 5:29 PM, Steve Jones wrote:

Are you sure?  If the transmitter defines the station location, then the remote transmitter in Maine wouldn't count for your DXCC here.  Am I missing something?
Yes. The current DXCC Rules say that all QSOs made by an operator from the same DX entity count equally. So I can rent a shack in Maine, run EU and AF and simply add them to the AS and OC countries I work from CA. It's been that way for a while. I'm guessing in response to some influential hams who moved and didn't want to start over.

LOTW is back up!

In case it didn't quite reach everyone, Microsoft has released a patch for MSIE last week.  It includes Window XP which was deprecated last month.  That problem allowed Flash to use the same memory space as MSIE, which created a gaping backdoor hole into the system.

It is STRONGLY suggest that this patch be applied; ALL versions of Windows; soonest.

Heart bleed has been around for a while, a known issue.  It simply migrated to the awareness of the press.  To suddenly be in a panic over these issues is common, but web designers have been working through this for some time.  If that is what the ARRL is doing now, it's overdue.

I don't believe that this is the heart bleed issue, the database access (via HTML, retarded way to process data) is offline.

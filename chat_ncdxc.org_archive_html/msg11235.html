> On Jan 24, 2016, at 12:34 AM, Jim Brown via Chat <chat@ncdxc.org> wrote:
> 
> I've been doing some poking around in the last few days, and found how to speed up Windoze by turning off some Windoze nasties that burn cycles and made my machine boot and run a lot slower.
> 
> First, the Windoze 10 Upgrade Nag is called GWX, and there's a ton of shit on your computer to make it work. To get rid of it, download, install, and run "GWX Control Panel" and follow the instructions in the Help (which is a blog by the program author). http://ultimateoutsider.com/downloads/
> 
> Second, go to Control Panel, then Action Center, then Change Action Center settings, then Customer Improvement Program settings, and if the Yes box is checked, click on No.
> 
> These two changes reduced my boot time by half. I have no easy way to tell if it's running faster, or by how much. :)
>

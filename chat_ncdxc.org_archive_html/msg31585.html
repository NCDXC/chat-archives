It's not F/H, MSHV allows for NON-DXpedition  responses in F/H format (two stations per stream).  And which stream is to be used.
I am never using MSHV on the bottom of the tone spectrum where DXped tends to hang out.
My intent is to clear the caller list as quickly and efficiently as possible so that others can use the band.  I never go beyond one stream so I'm not taking up inordinate amounts of spectrum. 

It seems a good compromise to me.

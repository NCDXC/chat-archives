If you want to run the highest quality run, once, I would run LDF4-50 from the shack to a junction box at the base of the tower.  Add a ground kit there and transition to either LMR-400 or RG-213 for the tower & rotator loop run.  I really like working with LMR400 from a mechanical standpoint vs RG-213, but there is some concern about dissimilar metals in the LMR shield that lead some to choose RG-213 as it is only copper and dielectric..  I have seen this most often in VHF+ runs that use a duplexer for a repeater, i.e. receiving and transmitting simultaneously.  Some avoid LMR due to the potential of diode formation between the foil and shield in LMR. 

We are probably splitting some very fine hairs here if talking frequencies below 50MHz, but I thought I would mention that difference.

Ideal run is bonded bus at the cable entry window for the shack with polyphaser -> LDF to tower base -> LMR400/RG-213 up the tower with additional ground kit at tower base -> service loop at the rotator loop so there is a pigtail that can be broken out at the antenna base to allow for later isolation or replacement of just the rotator loop portion of the feedline.  If we're getting very serious, a ground kit at the top of any tower run exceeding 20' isn't a bad idea, either.

If I was building my 'forever' tower, this is how I would plan it.

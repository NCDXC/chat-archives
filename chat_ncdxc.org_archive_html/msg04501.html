When you install new audio devices (sound cards), Windows makes the last one
that you installed the "default" sound card, so Windows will now be using it
for its sound and you may not be able to select it for your digital mode
software.  This is also why you sometimes hear Windows "sounds" on the bands
as people don't know that their computer is now talking to the radio.

I don't know which version of Windows you have but on XP, if you right click
the speaker icon in the system tray and select "Adjust Audio Properties", it
will open the Sounds and Audio Devices Properties control panel. Select the
"Audio" tab at the top and set the Default Devices to the normal device for
your computer, it may be something like "SoundMAX Digital Audio", the other
choice will be the one for your new sound card interface which will probably
be displayed as the default when you open the properties.

Then you should be able to reselect the sound card inputs for your programs.
I hope this helps.

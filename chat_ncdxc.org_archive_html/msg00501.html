On Nov 23, 2012, at 10:12 AM, Craig - AE6RR wrote:

> 
> I wish that Elecraft would do something like the way Kenwood radios work.  I have a lot of experience with Kenwood TS-2000 and TS-480s and I watch the main frequency display.  When you key up the display changes to the transmit frequency (the VFO displays are swapped).  It is very easy to see where you are transmitting.
>  

Similar on my Icom 7600. Both VFO frequencies are always displayed. The active VFO is in BOLD, and the inactive one slightly dimmed. If you are in Split, there is a giant white LED bar that displays "Split". There is also a "TX" indicator next to the VFO that shows the transmit frequency. When you key up in Split, the boldness shifts from the RX VFO to the TX VFO, so it is always obvious: (a) Whether or not you are operating split, and (b) Which VFO is transmitting and which is receiving.

(Also a nice add-on; you can "lock" the receive VFO so that if you are in Dual-Watch (listening to both VFO frequencies simultaneously), and are tuning to find the DX station's receive frequency, it will NOT inadvertently move the RX VFO if you are still turning the knob when you go back to his run frequency.)

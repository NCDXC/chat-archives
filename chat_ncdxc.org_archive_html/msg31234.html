I have some suspicion that Yuris has been using MSHV robot mode during the wee hours. Unlike F/H, which caps the number of signal reports to a single hound at three, MSHV has no limit. Hence, the endless reports. 
The advantage of F/H is that successful hounds get to use a likely clear channel for their R-SNR report to the Fox. In MSHV there is a possibility of the hound’s report not being heard because of interference from other hounds.
The US based DXps seem to be using F/H, and the Europeans MSHV. 
Just my two cents.

Al, they are using MSHV, a Bulgarian add on to FT8 software that allows multiple streams and somewhat emulates Fox/Hound without the requirement for Hounds to call above 1000 Hz.   It is very common in EU and many EU DXpeditions are using it.

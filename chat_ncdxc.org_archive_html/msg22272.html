I know a lot of NCDXC members like DXLab Suite, but for what it's worth, I prefer Ham Radio Deluxe.  It also handles all the features you listed (and more) with a lot fewer windows open.  For me, DXLabSuite always seemed like a loose collection of separate programs that may work together but were more work to manage.

With Ham Radio Deluxe (HRD) the Logbook program handles logging (obviously) and all the integration with LoTW, Clublog and QRZ, connection to theDX cluster of your choice.  The main window in HRD can be used to directly control your radio if desired (although I typically just minimize it and use the actual radio).  For digital modes, HRD has DM780, which opens as a separate window and can handle just about any digital mode except FT8/FT4.  HRD will integrate directly with WSJT-X for FT8/FT4, etc.

There is a free version of HRD (version 5.xx), but it's several years old and doesn't have all the latest bells and whistles.  The only support for that version is via online forums.

I purchased version 6, which includes bug fixes for life but not feature enhancements.  If you want new features and on-going technical support, you have to pay a fee each year.  I have received a prompt response to any technical support issue I've opened — they responded quickly to my support tickets.

One approach to spending less on HRD is to buy version 6 initially, and then don't pay the fee again until HRD releases some new feature you care about.  Some people find that works just fine for them.

In general, I'm OK with paying for software as long as it does what I want it to do and isn't exorbitantly priced.

I think both DXLab Suite and HRD are good options.  It really just depends on which one you like best — sort of like Ford versus Chevy — and whether or not you're OK with paying a fee for HRD.

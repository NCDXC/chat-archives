Ron,

I think I staggered out of the cave. Mustang is apparently still in Beta; my SDA-100 was delivered last November so it had Pinto loaded, and the 6m dimensions were FUBAR. The Mustang "Manual" that K4LDI attached was missing the illustrations and was useless. The instructions included re how to use Flip were somewhat obtuse/confused, and therefore so was I. It turned out that I did correctly install Mustang. What threw me was it coming up in DB18 mode instead of 3 el. I apparently forgot that you select your antenna from a Setup menu. I stumbled over that and then that was clear.

There are new features that are not explained, like Auto-Retract, which is not in Pinto. The use of the coarse/fine buttons in Setup is new. It is really easy to get confused and adjust the wrong thing, e.g. tuning the reflector instead of the DE. Thank goodness for Default mode. And I finally got the controller to "take" my changed 6m DE setting.

So I think I may be OK. Thank you so much for taking the time to respond.

Garry

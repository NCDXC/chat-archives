I have been using StorageCraft ShadowProtect for a number of years.  I use 2 or more backup devices, both USB (a pain on my operation system since I have to turn them off when I reboot) and LAN devices (always on).  Every couple of weeks or so (when I remember to do it) I move the backup from one device to another.  The idea is that I have the computer and last nights backup and if both of them fail I have one that is a few weeks old.

StorageCraft does one of three things:  Full backup, incremental backup and differential backup.  Full backup either when the operator requests it or if one is not available on the media.  Incremental backup includes only those files changed since the last incremental backup (i.e. two nights ago).  Differential backup chooses those files changed since the last full backup.

Incremental is fastest but requires all the incrementals to restore or create a disk image.  Differential is a bit slower, uses more disk space per backup but only requires two files to restore or create a disk image.

I have recovered individual files using a disk image (essentially the complete backup disk as a new drive in the file system).  It is mounted and dismounted by operator command.

Backup is on a schedule, mine is at 0230.  I found that my OS behaves better if I start up fresh each morning so I wrote a script to shut down the OS at the end of the backup.

Not Free.

Bob Van Tuyl, K6RWY

On Monday, June 1, 2020, 01:53:17 PM PDT, Richard (Rick) Karlquist via Chat <chat@ncdxc.org> wrote:


As long as we are on the subject, does anyone have a
suggestion for a backup tool that:

1.  Makes a bootable image of the SSD, not just file copies.
(IE:  a "time machine").

2.  Is free (or at least cheap).

3.  Works on (and has documentation for) Windows 10.

4.  Is able to write bootable thumb drives on its
own instead of simply saying how easy to it is to do
without any help from the program.  I suspect Microsoft
doesn't want it to be easy to do in the first place.

5.  Has a manual that is up to date and is not written
in broken English.

I have tried AOMEI and MACRIUM, without success.
In both cases, I never got past the USB boot drive
creation process.

A long time ago, I used to use (paid) Acronis.
It automatically backed up on a schedule and
the operator didn't need to be "religious" about it.

I only need a tool to mirror/restore my SSD.  I don't need
any of the other features they include for IT people,
such as cloning, partitioning drives, etc.

Rick N6RK

This looks like a great option for Windows.  I ran Hamclock on an old HP Pavillion laptop that I'd switched over from Windows to Linux Mint, and the best option I found to run a Hamclock-like display on this machine is https://dxlook.com/.  This is a website that will run on any browser/platform, and it provides the information I most valued on Hamclock -- MUF (I prefer the summary view) and SFI, A/K/SN indices. I put it in dark mode and run it in Chrome full-screen mode. Works great for me.Should run fine on a Raspberry Pi or Inovato Quadra running Chrome/Chromium.

Hamclock is a nifty piece of software that runs on a Innovato Quadra or other Rasberry Pi device. It is a very useful application in the Hamshack and it "looks cool"

 

Yesterday, it was announced on Facebook that the creator of Hamclock, Elwood Downey WB0OEW had died under unusual circumstances. He is now listed as SK on QRZ.com

 

Apparently, HamClock is dependent on Elwood's server for information. That server access will expire in June 2026. My Hamclock has already stopped working and I am unable to log into it to troubleshoot. 

 

This is a really sad ending to what has been a great product. 

 

Paul N6PSE

 

His website: https://www.clearskyinstitute.com/ham/HamClock/

 

States:  HamClock has reached end-of-life, the last release is version 4.22.

All HamClocks will cease to function in June 2026.
Thank you for your interest.

Elwood Downey, WB0OEW, ecdowney@clearskyinstitute.com

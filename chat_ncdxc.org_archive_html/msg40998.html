I live 6 miles from a 50 kW AM station on 1160 kHz.  The Hi-Z amplifiers are completely useless here.  They overload hopelessly ... and (for complicated reasons) there is no way to install a high pass filter to protect them.  Even if you have no strong- BCC QRM, IMHO the design is poor because you have to tune up the included band pass filter to whatever frequency you want to listen to.  This is bad because you now can only use it on that frequency.  Also, the bandpass filter will introduce phase shifts that mess up the pattern.

What I did instead was to use 30 high verticals made of irrigation pipe, and connect the vertical to the phasing network using a 9:1 impedance transformer.  Worked for me.  YMMV/

> The first sentence below is true if the K3 CONFIG parameter SPLT SV is set to NO (the default).  However, if you want the radio to retain split status as well as RIT/XIT status when power-cycling or changing bands, then you can set SPLT SV to YES.  This flexibility allows the radio to be configured either for DXpeditions/contesters or for very different applications where the operator doesn't want to be surprised by split mode after a power-cycle or band change.  The K3 has an extremely diverse customer set, more so than other radios, thus the flexibility built into the CONFIG capability.  The downside is your comment "takes some getting used to" when you come from another radio that has a single way of handling a specific part of the user interface.  In other words, there are many K3 customers who would be troubled if split status were saved through a power cycle or band change.  Thankfully, the K3 can handle both groups of users.
>  

What would appear to be useful is a Configuration Wizard application, running on a computer connected to the radio. Rather than require users to drill through every menu, and interpret the sometimes cryptic display (due in large part to resolution limitations) while holding the manual open on their lap, an app could walk the user through the various configuration parameters, EXPLAIN what each one does, possibly offer advice as to reasonable settings for different applications, and then write all of this to the radio. It could also save the parameters in a named file, which would allow users to store different parameter settings for different uses (e.g., a radio used by multiple operators, or one used both at home and on DXpeditions).

The key is not just having the flexibility and the ability to configure the radio to one's liking (which I know the K3 has), but to understand which settings do what, and when a change from the default might be appropriate. 

Or does this exist already? I don't own a K3, so I don't follow the support discussions. (It's well known that I'm not a K3 fan, even having used the radio numerous times, primarily because of the difficult and cryptic user interface. I find that the large, high-resolution color display on my Icom 7600 allows much more information to be presented, and in a more intuitive manner than on the K3's limited screen. I still occasionally pull out the manual, but it's quite rare, and typically for only the most esoteric information, like the encoding of CAT commands, or the pin assignments on connectors.)

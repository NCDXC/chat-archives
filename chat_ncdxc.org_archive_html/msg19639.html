I received a few emails asking me what the FT4 frequencies are. Apparently they don’t show up if you upgraded from an earlier version of WSJT-X.

If you select FT4 as mode in WSJT-X and you don’t get a new list of frequencies for this mode then you can update the frequency listing in the settings. 

Go to :

File -> Settings -> Frequencies (TAB) 

Right Click into the “Working Frequencies” list and select “RESET”

Now you should be able to see the FT4 frequencies suggested by WSJT-X.

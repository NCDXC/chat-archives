(My shack computer boots up and go straight into user mode without requiring a user login.)
In Win 10, EITHER press 'logo + r" or press the logo key, type 'run' to call up the run app, then type "shell:startup" in that window to open a file explorer window.
Note that location it opens to, then copy the shortcut of any program/s you always want to start at boot to that location. Done!
For software I want running ALL the time (my weather collection/posting app) it's very handy.

Paul,

All members of the NCDXC are welcome to use our Guys-Up-North Discord server.  In this server I have DX Network and RBN spots piped into channels for DXPedition or rarer DX alerts.  We have had the DX-Chat channel that gives us a voice channel for discussing DX, and we also call out in real-time QSX frequencies.  There are several NCDXC members already participating.  The information on the Discord was in a recent SCDXC newsletter.

http://www.scdxc.org/docs/newslist/newsletter_20230600.pdf
   (on page 15-19)

The name Guys-Up-North (GUNs) was given to a previous version of our DX-Chat we have been using since 2015, so please don't be discouraged by the name of the server.  Invitation links are time-limited invitations, so if you are interested in joining us, please send a request to me using the email address in the NCDXC membership list. 

We even have a tech-support channel for helping users setup their stations when they run into problems.

73, Bill
N6WS


On 6/24/2023 8:10 AM, Paul N6PSE via Chat wrote:
I would like to start a discussion on the best way to alert for rare/exciting DX.

Currently, many of us use NCDXC email "chat" . This works well for me as I can monitor my email while watching TV and it does not bother me when I am sleeping. 

Recently, a group started using Discord. Discord is heavily used by Hackers/Gamers and has a terrible reputation for information Security. 

Others are using Mumble, while others are using Slack.  Slack is pretty good but you have to remember to mute it when you go to bed. 

It would be great if we could all use one platform to alert each other of rare & exciting DX. 

What do you think?

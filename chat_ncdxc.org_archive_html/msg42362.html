For clarification, the FM on 2m is not going away, it will be supplemented with a DMR “IP connection” which will be RF free. Meaning, we are not looking to install a 70cm repeater as originally planned. It will be only accessible via the internet if you want to use DMR. There will be an NCDXC TG (Talkgroup) Why DMR? It is readily available, easy to use and will allow our members outside the repeater range to be able to communicate with us and/or hear us. Allstar is ALSO being looked at and will be used at least for voice forwarding application which is to be officially announced later. This will integrate into slack. There is a good chance the using the same device we are looking at we will be able to establish BOTH. Stay tuned for more information!

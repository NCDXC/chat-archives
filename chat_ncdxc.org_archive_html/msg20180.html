Hi Rick,
15 years ago I was using Norton Ghost to quickly back up and restore entire Windows disk partitions. Now I’m on Mac, but it's not relevant for Windows.
What should always work is to make a bootable USB disk with your favorite Linux distribution and use that to back up or restore partitions or disks. Once you boot it up, a simple, short command line is enough to quickly fix (or thoroughly wreck) a disk. There must be GUIs/apps making it easier.
And I guess this is article is useful if looking for a Windows-only solution: https://www.techspot.com/guides/1721-clone-backup-usb-drive/

On Monday, June 1, 2020, 01:53:17 PM PDT, Richard (Rick) Karlquist via Chat <chat@ncdxc.org> wrote:


As long as we are on the subject, does anyone have a
suggestion for a backup tool that:

1.  Makes a bootable image of the SSD, not just file copies.
(IE:  a "time machine").

2.  Is free (or at least cheap).

3.  Works on (and has documentation for) Windows 10.

4.  Is able to write bootable thumb drives on its
own instead of simply saying how easy to it is to do
without any help from the program.  I suspect Microsoft
doesn't want it to be easy to do in the first place.

5.  Has a manual that is up to date and is not written
in broken English.

I have tried AOMEI and MACRIUM, without success.
In both cases, I never got past the USB boot drive
creation process.

A long time ago, I used to use (paid) Acronis.
It automatically backed up on a schedule and
the operator didn't need to be "religious" about it.

I only need a tool to mirror/restore my SSD.  I don't need
any of the other features they include for IT people,
such as cloning, partitioning drives, etc.

Rick N6RK

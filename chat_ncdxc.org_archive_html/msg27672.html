Bob,

If not enabled, turn on one of the logging destinations (same menu, higher up).

Rebuild database.

If desired, disable the logging feature (WSJT-X puts the log entry in for me, if also enabled in JT Alert, the log is double entered, my JT Alert log is UNchecked, off).

You can also import a log, then rebuild the database (same way as above).  That import can be the FTx log/s (ADIF) or your main logbook adif.

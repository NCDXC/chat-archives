On Thu, May 30, 2024 at 7:05 AM Rick Bates, nk7i via Chat <chat@ncdxc.org> wrote:

Presented to another group, yesterday...

The first rule of ransomware, is that you don't talk about ransomware (when involved by it) for ANY reason.

  • A forensic autopsy must determine HOW the network was crippled; including WHEN (so the validity of backups can be known).  WHO isn't even on the radar (since access knowledge is often shared and not much recourse is possible).
  • The hole/s or means of that access sealed tightly.  Then...
  1. Pay the ransom; hope it's a one time payment AND that the decryption key is valid (IF sent).
  2. OR rebuild based on backups;
  3. IF there is a backup that isn't affected; restore to the last best known state (and verify the holes being closed)
  4. On the same hardware (owned) or on a service providing upkeep and maintenance (costs money) like AWS (where arrl.net, arrl.org live).
  5. If no backup, a total network and server rebuild could take months (with a lot more apps using it, modified to work again).  If this is the case, hopefully (besides lessons learned) it is made simpler and more efficient (for user and internally).

That the ARRL has not been saying anything of content, is telling by itself.  No mention of restoration timing has been said.  

That two weeks has passed tells us it's not a simple restore (which would take mere hours) and that it's not hardware related (easily replaced).  We don't know if there is a (valid) network backup (or server for that matter).  There is a LOT we don't know (but a LOT of rumors abound) while we hope that the log data is secure (and uninfected).

We can slam the ARRL (I'm a member but no fanboy) for not being current in security, no communication or anything else; but that is pointless for now, what's done is done.

The loss of decades of logs would be immense (silent keys, cards now lost through any means, heirs deleting logs).

There is little we can do but wait and see how the ARRL responds/reacts.  Let's not raise the noise floor here unnecessarily; we're all already frustrated by this more than enough.

Whining, won't help.  WE have no option other than wait unless one wants to dabble in the politics of the board (yeck!).

While we hope for the return of LOTW.

TU es 73,
Rick nk7i

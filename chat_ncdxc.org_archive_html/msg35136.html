The release said that there was no loss of signal strength despite the multistreaming. Both MSHV and WSJT F/H suffer a 6 dB loss when transitioning to 2 streams, and an additional 3 dB for each additional stream. The release text mentioned a 1.5 KHz envelope that could address up to a whole bunch of Hounds without the weak signal decoding issues we see now. 
This is huge, because the QSO rate-limiting step for DXps is the need for the Fox to limit the number of streams to remain strong enough for decoding. On H40WA there were times we could only single stream. 
I’d enjoy learning more, and sure hope I interpreted the announcement correctly. 
And Paul is right on, as usual!

A while back I sent out an invite to this Chat email list - to join the Heard Island / VK0EK Project. Several of you had signed up, and I thank you for that - and I look forward to seeing you this Saturday. If you'd like - please check out the Heard Island Project web site:

http://www.heardisland.org/

If you are interested in joining one of several "local work parties" for this project, I would still like to extend this offer. I say work parties, because signing up and being listed on the Heard Island project web site will require attending meetings and volunteering to do actual work. Some of the work parties include:

* Fund raising
* Warehouse work at the Cordell Expeditions Richmond warehouse - packing, cleaning, moving, loading, unloading
* Testing DXA2 - satellite link and data links
* Testing gear and antennas
* Go shopping
* "Own" other parts of the project - TBD
* Attending what will seem like a weekend long "field day" of sorts - not really a field day - but a weekend where we set up the structures that will be used on Heard Island, and the gear, power, etc needed to run things

I have "Table 27" set aside at the Visalia IDXC Banquet for anyone on the Heard Island Team to sit together - if you like. 

Again - this is hands on work. If you are interested, please let me know. I will give you the details of where and when this Saturday meeting will adjourn. We are especially interested in local, Bay Area DX-ers since its most practical to get together in person - and often.

73,

Rich

Kenwood TS-990S owners – Our 5+ year old radios got a new lease on life.  After years, Kenwood released a new FW release 1.20 and 1.21(bug fix).  Many new features, nine of them.  See attached.  It is a big one taking 12 or so minutes for the radio to install them.  Many new features and improvements.

 

Find it at https://www.kenwood.com/i/products/info/amateur/software_download.html.  Also update the ARCP so it is compatible with the new TS-990S if you run remote.

 

BTW – remember don’t un-ZIP the file.  The radio does that.  Start the radio in update mode (hold M.In and Power up).  If attached a PC it will display a dialog box – directory type.  Drag the ZIP file to the box and sit back.

 

If using a Thumb drive put the ZIP file in the root directory and plug it in and let it go.

 

BTW – With the scrolling waterfall the preselector bandwidth is really obvious and track the VFO.  Have the display in Fixed Mode showing the whole band.

 

I have attached a little easier to read 2-page Kenwood update description.

 

Enjoy,

 

Bob – W6OPO

Supporting Jim's choke message -

If using coax on the RX antenna feed and if you think you will have too much
RF getting onto the coax's shield then a choke is a good idea.  Put it as
close to the radio as possible.

I never found a choke necessary.  If it isn't broken I don't fix things.

If using balanced line that is indeed balanced then of course a choke is not
used.

When I uploaded a WPX log this summer, I got the following response from Club Log for a QSO with CQ8X:
CQ8X was an operation on 2012-03-24/25 by OH2UA using the Radio Arcala CR2X station.
OH2UA operated the station remotely from Finland (tnx OH2BH). QSOs with this station are thought to be not valid for DXCC.
Is this true -- that a remotely operated station is not valid for DX awards? If so, what is the rationale?

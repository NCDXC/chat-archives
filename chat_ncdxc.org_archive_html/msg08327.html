On 11/10/2014 10:58 AM, Andreas Junge wrote:
Garry,

I am assuming you are still using DX4Win :-)

Go to File-> Import and select ADIF.

Select the lotwreport ADIF file you downloaded from LOTW. 

Then you will see a dialog box with the title “Options for duplicate QSO’s”. There select “Imported QSO sets confirmed flag”.

Hit OK and you are done. Don’t forget to save the log file. 

73, Andreas, N6NU

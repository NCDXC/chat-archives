On 12/4/17 at 12:03 PM, chat@ncdxc.org (Harry Makler via Chat) wrote:
As I do, who has a Kenwood 590SG, an Apple (IMac) computer and has successfully installed and uses FT8 ?
One item that is very annoying is that it is possible to have two drivers installed for the FTDI USB <--> RS-232 interface converter. If there are two drivers installed, then Hamlib will not be able to initialize itself. To find out if you have more than one driver installed, in terminal type:
Williams-MacBook-Pro:~ frantz$ find /System/Library/Extensions -name *FTDI* 

When I got the result:

/System/Library/Extensions/FTDIUSBSerialDriver.kext
/System/Library/Extensions/FTDIUSBSerialDriver.kext/Contents/MacOS/FTDIUSBSerialDriver
/System/Library/Extensions/AppleUSBFTDI.kext
/System/Library/Extensions/AppleUSBFTDI.kext/Contents/MacOS/AppleUSBFTDI
I removed the /System/Library/Extensions/FTDIUSBSerialDriver.kext file (by moving it to another location), rebooted and everything started working. 

Perhaps this will help.

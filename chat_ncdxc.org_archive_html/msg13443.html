You guys are ahead of me because I haven’t actually tried it yet, but it “seems” like a very nice upgrade.

 

DXKeeper allows automatic uploads to ClubLog every time a QSO is logged in DXKeeper, if you configure it to do so.  But, there is, by design, no way to automatically upload to LotW.  (Dave says that would be too much of a load on LotW.)

 

However, with this new ClubLog-LotW feature, ClubLog can automatically upload to LotW, so it should no longer be necessary to manually upload from DXKeeper to LotW.  If I understand it correctly, this is a nice benefit.

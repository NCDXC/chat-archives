… and to save anyone else floundering around as much as I did this morning, here are some slightly more explicit instructions:

 

  1. Download, install and run Google Earth Pro, if you don’t already have it.  There’s a link from Michael’s blog.
  2. Download and unzip the KMZ files (from the links also on Michael’s blog), saving the unzipped data files somewhere handy on your computer e.g. C:\Radio
  3. In Google Earth Pro, from the File menu click Open then navigate to either KMZ file and open it.
  4. Oooh and aaah as you spin the globe to see where the lines go.

 

I guess we can probably also load the KMZ files into plain Google Earth (the online/cloud version) but I haven’t tried that.

 

Thanks Michael and Francesco!

Vessels that use Winlink have a acceptance list so sending an email through winlink
without being on his acceptance list will result in a bounce. There is a way around this.
System operators sometimes need to send to a user that hasnt included them on their
acceptance list, they do this by sending a SYOP Service message message.
 
The normal Winlink address for a vessel is callsign@winlink.org  (WH7XH@Winlink.Org)
End the subject line of the message with   //WL2K and the system will deliver the message
as a Service Message otherwise your message will bounce.

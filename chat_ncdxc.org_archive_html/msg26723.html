I have a used 3 element SteppIR antenna for sale, posting for NCDXC org before I list it anywhere else.  Here's the story.  I bought it used, have never tested it (though I supposedly purchased a working set), with the intent of restoring and using but ended up getting a 40-10m SteppIR later and also ended up with N6TA's SteppIR so I no longer need this project.  Here is what is included:

 

-Boom plate and clamps

-All the fiberglass telescoping elements.  Used, if I were to keep I would repaint but they are not that bad as is.  The elements are currently collapsed.

-One driven motorized element with cord attached.

-One reflector motorized element with cord attached

-One director motorized element with cord attached.

-One extra motorized element which could be used as passive or driven element (has the so-239 connector)

-One control box with extra ROM if you were to make a 2 element out of this

-Connector tube which hangs on the boom

-bolts and hardware for the boom

-complete boom, currently in pieces

-one caveat to the boom;  one of the sections was bent;  I have an exact size replacement for this to be included, you just have to drill the holes

-all the rubber connectors for the elements

 

What it doesn't have is the heatshrink tubing for the elements, silicone tape, and the connectors for each of the motor controls and where they come together on the boom.

 

Asking $500

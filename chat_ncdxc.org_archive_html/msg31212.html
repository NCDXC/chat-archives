The normal FT8 transmission actually transmits twice for redundancy.  They give up the double message in order to send one after another, less reliable weak signal coms but more Q's.

On Tue, Oct 10, 2023 at 10:28 PM Paul N6PSE via Chat <chat@ncdxc.org> wrote:
Al, they are using MSHV, a Bulgarian add on to FT8 software that allows multiple streams and somewhat emulates Fox/Hound without the requirement for Hounds to call above 1000 Hz.   It is very common in EU and many EU DXpeditions are using it. 

There is also a feature called "Robot mode" where you can tell MSHV to answer callers while you shower or take a nap. I've seen a lot of evidence of robot mode lately.

All true, Steve, except that BItlocker is on by default on Windows 11, as is OneDrive.  And also by default, Windows will store your Bitlocker key on OneDrive.  And while it isn't impossible to turn Bitlocker and OneDrive off, Windows 11 makes it extremely difficult... much more so than on Windows 10.

I consider myself a pretty knowledgeable PC user, and one who has used Windows from the very beginning.  I've built my own systems, rooted around in the Windows registry when necessary, and generally figured out how to get Windows to do what I wanted when I wanted.  But with Windows 11, the level of effort necessary to keep Microsoft from virtually "owning" control of your PC is just not worth the effort.  I've played with Linux off and on for years, mostly with regards to Raspberry Pi's, but it seems like it's time to make a full switch to Linux and give Windows the heave-ho.

At the time we were considering the "repeater" site for several purposes in
addition to only use as a repeater site. If we were only using the repeater
site for the repeater that is exactly what we would have done. We may yet do
just that. We initially included the 120V AC inverter in the system design
because the equipment we need to make on site measurements of the repeater's
performance and compliance with the FCC and with NARCC only runs on 120V AC
and the closest 120V AC is miles away..or via a portable generator. One of
our repeater monitoring instruments for example is a beautiful old beloved
Cushman CE-5 "COMMUNICATIONS MONITOR" and it is large, heavy and won't run
on anything but 120V AC. We also view the repeater installation as yet
another backup emergency resource in a wilderness area if we need one and an
alternative radio site (and an emergency off grid power resource) for Field
Day and other club events.

73, Ted, K6XN

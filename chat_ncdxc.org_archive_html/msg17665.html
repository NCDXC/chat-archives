I just discovered a fantastic 'companion program' to Wsjt-x that might not be widely
known yet:

Gridtracker (https://tagloomis.com/)

This program has numerous options to visualize data received from FT8/FT4 
in realtime. I have only scratched the surface of this program - it seems to have 
integrations with common logging programs as well. 

I installed it on X86 64bit Ubuntu 18 LTS and Raspbian buster (running on the new 
Raspberry PI 4).

NOTE: On the Raspberry PI - Gridtracker has to be launched with the command:

$ ./GridTracker --disable-accelerated-2d-canvas --num-raster-threads=2

on Ubuntu it is just:

$ ./GridTracker

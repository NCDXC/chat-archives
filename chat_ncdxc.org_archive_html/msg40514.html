Yesterday I looked at "Windows Update" on my shack computer. Some time ago I had turned off automatic updates. Of course it had a red warning that it was out of date. I did "check for updates" and various updates started to download. Anyway, I was eventually asked "OK to restart?", which I did. I continued this cycle and then let the shack computer run overhight. The next morning I enabled restart and it installed more updates. At that point it displayed at green arrow saying it was up to date. But it did NOT offer the ESU plan.
THEN, I logged into my Microsoft account which I almost never use. I had never logged into it from the shack computer. After doing that, I saw that the windows update page only now displayed a message in the upper right corner saying I was eligible to enroll in Extended Security Updates. I clicked on the button and it asked me to log into Microsoft again and then it displayed a button to push to enroll. However, for whatever reason it displayed "$0". I clicked on that and now the Windows Update page displays a message saying I am enrolled to receive ESU's.
That's it! Worked for me... And the price $0 was right. (By the way, the shack computer is NOT eligible for W11). 

Good luck

It has been a while, but our club's Echolink node is back on the air.  After some time, I gave up chasing low-level digital noise that causes the node to falsely trigger a user to transmit for a quick second every so often.  I even swapped out the radio with no improvements.  So the noise may be generated by the Raspberry Pi itself.

Please, feel free to connect to N6NKT-L.  The node allows the user to connect to our 2-meter repeater.

In the meantime, I will build another Raspberry Pi Echolink node and swap it in some time in the near future.

Please, feel free to check out the node and if you have questions about it, drop me a line to N6NKT@yahoo.com

73 de N6NKT, George w.

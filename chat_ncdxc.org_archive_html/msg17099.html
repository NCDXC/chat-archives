If you are a Firefox user you may have found all your browser is not acting normal today. It is because add-ons no longer work due to a certificate that expired at midnight GMT. This is a complete screw up on Mozilla's part - there is nothing you did other than trust the Mozilla org to actually know what they are doing...

How to fix in the "short" term: go to https://blog.mozilla.org/addons/2019/05/04/update-regarding-add-ons-in-firefox/

Until the hot fix gets distributed as discussed in the link above, Firefox is unusable if you use add-ons (pretty much everyone).

I still recommend Mozilla as the safest, most secure and most private web browser *if* it's properly configured with the right set of add-ons.

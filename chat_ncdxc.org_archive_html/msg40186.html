> I like the suggestion that there would be a separate award for working all “currently available countries”. 
> 

Which only raises the question, “What makes a country available? (or unavailable)?”

Is North Korea available? Ham radio is banned there for its own citizens, but a number of foreigners have received temporary permission to operate, and there have been QSOs confirmed within the past decade.

How long does it take since the last QSO for a place to be perceived as “unavailable”? Does a place become unavailable only due to political/legal considerations, or because it is virtually impossible to get to, and no one has been there in a long time?

Some countries allow ham communications IN GENERAL, but prohibit communications with specific “enemy” nations, e.g. Israel. This used to be true for American hams with respect to China, Libya, and others. Are these entities “available” in the places that allow such communications, and “unavailable” to the ones that don’t?

“Worked All Available” isn’t as straightforward as it sounds.

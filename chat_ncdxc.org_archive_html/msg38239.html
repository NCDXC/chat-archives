WSJT-X and WSJT-X Improve share the same code base. Both are built on the same servers.  WSJT-X Improved is a full supper set of WSJT-X and has all the features of WSJT-X. WSJT-X Improved gets / has all the fixes as WSJT-X. WSJT-X Improved is where they test new features before adding them to WSJT-X. but not all WSJT-X Improved features will be added to WSJT-X in future WSJT-X release.

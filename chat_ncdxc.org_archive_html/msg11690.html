Rick,

I believe it doesn't count for any country right now.  You only can count
the prefix as being worked.

This is the answer I found on the web for now:
"This same issue of the ARRL Letter also states that contacts made before
the Event Date will be counted as Serbia for DXCC purposes. However, a
subsequent email from Bill Moore, dated 2012-09-24, specifically states that
QSOs with stations in Kosovo using the Z6 prefix do not count for any DXCC
entity. Sep 12, 2014"

Probably it's best to call the DXCC desk back at ARRL for a more up to date
answer.

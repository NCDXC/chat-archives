I took delivery of 4O3A's new 8x2 Antenna Genius last month and put it on the bench before installing it in the station to replace their earlier generation 6x2 switch. For isolation between radios, the older switch beat my old WX0B unit by about 20 dB. The new one is 20-30 dB better below 15M, but not above 17M. Also, return loss increases with increasing frequency on the new switch, and is not acceptable (to me) on 10M and 6M. I've made some suggestions to improve these performance issues, but they won't be easy to implement without some redesign.
That said, the new unit is VERY good -- I measured isolation of nearly 110 dB on 160 increasing to about 90 dB on 20M. Isolation depends on which port is selected. So does return loss. BTW -- I measure isolation with only the two radio outputs terminated, because I believe this is more representative of real world conditions than terminating or shorting unused outputs (unless the product does that by switching). Real antennas are usually only 50 ohm sources near their design frequency, and vastly different everywhere else.
This is a very new product, and the only doc is an incomplete version written for Flex radio users. Except for the issues that impact the shortcomings noted, construction is excellent. Automation features look quite versatile, multiple units can be used together and controlled via wired Ethernet. They also work with relay pulls from an external decoder, and from some forms of BCD data. 

73, Jim K9YC

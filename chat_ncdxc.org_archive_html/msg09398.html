The shortcoming of this arrangement is that because the choke is on the bottom of the feedline, the feedline can carry common mode current in the dipole mode because of feedline length. The common mode current would depend on the balance of the antenna. The early slides in http://k9yc.com/CoaxChokesPPT.pdf show conditions that can cause common mode current.

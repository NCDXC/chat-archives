On Feb 6, 2014, at 9:29 AM, Bob Lanning wrote:

> No choke is needed on an RX antenna.
>  
> As for isolation I have found the TR relay in the radio provides all that is needed.  Radios with a dedicated RX antenna port not only open the RX antenna port when in TX mode but also short it out.
>  

That is correct, but remember to be careful if you use the RX antenna port on a transceiver while using a DIFFERENT radio as the transmitter. I have some vintage rigs here (separate Transmitters and Receivers), and occasionally use a modern radio (my IC-7600) as the receiver while transmitting with a boat anchor. There is a T/R Relay in the boat anchor, but that doesn't short-out the RX antenna input in the receiving radio.

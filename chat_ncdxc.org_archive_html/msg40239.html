Matt & Co.

 

Full disclosure, the only award I ever applied for was as a Novice, the Ragchewers award.  The pursuit of DX has been strictly personal, competing against myself, so anything I say here can be taken with a grain of salt as I'm not really qualified to comment.  Having said that, I have worked them all.  I started in 1968.  Yes, it took a long time, like 34 years.  However, I don't think #1 Honor rules should be changed.  Why should an award be obtainable?  I didn't think this would be obtainable in my lifetime either.  There are many obtainable awards along the way to this top spot.  I've seen many changes in entities since 1968.  Most athletes for example are never able to attain Olympic status.  Why water down the top spot that many took their whole lifetime to achieve.  There seems to be plenty of awards available for all and do we really want the top award to be obtainable by all?  Is there value in that?  All this discussion is making me think I should apply before I die.  Countries like North Korea will surely be available in the next 20-30 years along with other changes if we wait.  In any case, I would leave #1 Honor roll where it is and if needed add an award that provides incentive, still hard to obtain, that would satisfy more of the masses.

respectfully,

I believe that if the ARRL offers and promotes an award that the Award should be attainable. Currently it is not possible (in a lifetime) to attain Honor Roll for anyone starting the journey now or in the recent past. Change is required when a program becomes stale.

 

Gene K5GS proposal to have an inactive list makes the most sense. It only requires a rule change and does not delete any entities. However, the outcry from the old timers of “NO CHANGE” is very emotional.

 

Compromise proposal: #1 Honor Roll rules remain unchanged and in effect becomes the legacy Award. You need to “Work them All” to receive the #1 Honor Roll status. No changes to the current rules for #1 Honor Roll. The rule to count entities for Honor Roll be changed to “you must have a total confirmed entity count that places you among the numerical top ten DXCC entities currently active in the last twenty years on the DXCC List” The Honor Roll has never been a “Worked them All” award. Honor Roll has always been an award for diligently “working the greatest amount of them but not all”. Additionally, a new award “High Honor Roll” could be created for working all currently active entities in the last twenty years.

 

In summary you would have three award levels (#1 Honor Roll, High Honor Roll, and Honor Roll).

 

Matt K6EME

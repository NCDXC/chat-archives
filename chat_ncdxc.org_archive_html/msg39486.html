A printed code is equally as valueless as the home made card could be.

I make my own cards to send, but each one is hand filled out and signed, which has been enough to be accepted.

ANYONE with a few simple skills can print out a card, from any entity, how can such a card be verified?  Electronic (emailed) cards are just as utterly worthless and go in the trash bin; anyone can fake them faster than printing one.

The station initiating that email, proved they have internet access and has enough savvy to create an emailed QSL, so they have enough skills to use LOTW (THE accepted standard of the day) which is accepted everywhere (even though it can be a PITA to set up for some places).

Still, some refuse LOTW for their own reasons BUT send a real QSL card; which is accepted AFTER verification with a card checker (another type of PITA here, the nearest is an 8 hour drive, each way).

Expecting the ARRL to change, won't happen as they are completely unencumbered by progress.

Yes, the method described below technically removes the DXK LotW upload step, but it also adds the analogous step in Club Log.  Same amount of manual work, just in Club Log instead of DXK.

 

I’d rather do the steps in DXK because (1) that’s my master log, not Club Log, and (2) I’m already in DXK doing my logging so it is easy to remember to do the LotW upload.

 

This new LotW integration in Club Log is more useful for people not using a logger like DXK that already has excellent LotW integration.  The feature’s main value for me, as a DXK user, is keeping my Club Log DXCC data sync’d with LotW and my DXCC credits.

You guys are ahead of me because I haven’t actually tried it yet, but it “seems” like a very nice upgrade.

 

DXKeeper allows automatic uploads to ClubLog every time a QSO is logged in DXKeeper, if you configure it to do so.  But, there is, by design, no way to automatically upload to LotW.  (Dave says that would be too much of a load on LotW.)

 

However, with this new ClubLog-LotW feature, ClubLog can automatically upload to LotW, so it should no longer be necessary to manually upload from DXKeeper to LotW.  If I understand it correctly, this is a nice benefit.

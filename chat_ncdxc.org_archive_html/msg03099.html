Spot collector does it DXCC way, i.e. bands and modes, not bandmodes. E.g. if you have P5 on 21 MHz CW and 28 MHz SSB, according to DXCC rules you won't need it on 28 MHz CW or 21 MHz SSB. But RTTY on either bands would alert.

There's program called SpotSpy you can buy and it is using SpotCollectors' data. http://www.dl9ho.de/SpotSpy/SpotSpy.htm. They call this Quixote Quest.

I'd start with the simplest.  Presuming that the computer power went COMPLETELY off for a time (> 2 min); unplug the radio USB (listen for the OS sounds, if any); wait ten seconds then plug it back in (listen for the OS sounds finding it).  If you hear the OS on connection, then search again for the CODEC.

If the computer was NOT completely down for > 2 min (time enough to completely turn off all things inside), then start there.  Sometimes USB devices get a 'stuck bit' that makes them finicky; a stone cold start can fix that.

You MAY have to restart the app/s using the CODEC, since some only 'discover devices' at start up of the app (DOH!).

Then report back any findings and the collective will get you on the air again.

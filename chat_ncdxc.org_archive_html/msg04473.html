IC-7800 users –

 

The long awaited FW upgrade to the IC-7800 is now available.  Nice improvements and many added features like a bandscope waterfall and more.  Another I like is a PC-less remote capability.  That means the IC-7800 has the Server function built in.  The RJ-45 jack is now usable.

 

http://www.icom.co.jp/world/support/download/firm/IC-7800/3_00/

 

http://www.icom.co.jp/world/support/download/manual/pdf/IC-7800_ENG_FirmUpInfo_V300.pdf

 

Kenwood – we sure would like that built in Server in our TS-990S’.  Still have to us a PC.

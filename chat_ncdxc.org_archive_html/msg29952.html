On Sat, Jun 24, 2023 at 9:52 AM Al N6TA via Chat <chat@ncdxc.org> wrote:
Paul IBU introduced me to HamAlert a couple months back and i like it so far.  You set a trigger for pretty much anything and with many conditions.  I have one for 5A1AL on the bands and modes that I want to work Abubaker.  The big caveat is that HamAlert needs to get the info that that condition is met.  It uses the DX Cluster, Reverse Beacon Network or PSK Reporter as data sources.  So, if an alert is issued by HamAlert, the whole world can know about it, too.  I have it give me a push notification and it can announce as 'DX' in cw if you want to know that it is a ham related notification.

Paul suggested there be discussion as to what DX should be put out as a spot.  I think there should be guideline for this and not a rule.  I try to spot what I think most DXers would bel interested in working, either as a possible new slot/band/mode/atno or as something that is fairly uncommon the west coast.  I see spots that I wish I did not need to click thru because they are just not interesting to me.  Like a TF on 20 M any mode.  If it is something an active ham would stumble on at some point in time, those that need it will find it and should not need spotting help to work.  They should be doing some band tuning anyway, IMHO.

Discussion on any DX topic works fine on this group and I see no need for something new.  If a new app offers some features that could be of interesting to most, someone please make a list of the features or provide a little more meat to consider.

73,
Al  N6TA

I would like to start a discussion on the best way to alert for rare/exciting DX.
 
Currently, many of us use NCDXC email "chat" . This works well for me as I can monitor my email while watching TV and it does not bother me when I am sleeping. 
 
Recently, a group started using Discord. Discord is heavily used by Hackers/Gamers and has a terrible reputation for information Security. 
 
Others are using Mumble, while others are using Slack.  Slack is pretty good but you have to remember to mute it when you go to bed. 
 
It would be great if we could all use one platform to alert each other of rare & exciting DX. 
 
What do you think?

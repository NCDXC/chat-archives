Hi Peter,

The interesting thing about Wine is that there is no Windows there.  It's a compatibility layer, instead of a full-blown virtual machine like VMWare or Parallels.  As such, there's no Windows version, and it's very lightweight.  And as I mentioned, the application windows behave sort of like regular Mac OS windows.  I don't guess every application out there is compatible, but most ham stuff is simple enough to work OK.

I've had very good luck with the Prolific PL-2303 chip.  I recommend you get the most generic one you can, because it'll probably be built straight off the application note and therefore be very well supported.  $10 or less should do it.

Supposedly the FTDI chips are even better but I've not used them.

Bear in mind, to get the port to work under Wine, you actually have to install the Mac drivers and configure the serial port on the Mac.  You'll also need to use the command line a bit; nothing too serious.  I've been meaning to post my notes about how to do all this stuff on the web.  Maybe this will finally be the impetus.

The MAC option is certainly viable, but if you don't want to buy expensive new hardware right now, you might try out (gasp!) Linux.   My XYL has all Apple stuff, and I keep a Windows box alive to run my Flex-3000, but everything else I've been very happy doing on Ubuntu Linux.   Been almost 20 years now since I defenestrated.

You can try it out on your current hardware, without disturbing Windows at all, by putting Ubuntu on a CD or USB memory stick, and booting your PC from there.  Tons of online support.   If you want to know how to <do x> just Google "ubuntu <do x>".

It will be somewhat slow running off the USB/CD memory, but you can see how it feels to use it.  If you like it, you can either install it over Windows and never look back, or install it beside Windows and have your choice of OS every time you boot up your machine.

So, as the saying goes ... Free.   No obligation.  No need to buy new hardware.   No time limit, just download and try it out. 

See http://www.ubuntu.com/download/desktop/try-ubuntu-before-you-install if you're interested or curious.

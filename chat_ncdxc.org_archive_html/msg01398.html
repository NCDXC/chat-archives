All are welcome to use the following new DX cluster node. It is fully connected to the cluster network and all spots sent to it ARE forwarded to the network.

Host: dxc.ncdxc.org
Port: 8001

The default user filter accepts DX that is spotted in CQ zones 3, 4 and 6. Users can change the filters.

This is a different port and node than the old west coast cluster.

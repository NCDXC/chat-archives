Well, it is time to try the new Digital Mode, FT4.  FT4 is the contest version of FT8.  If you know how to use FT8, you are ready to try FT4.

Each transmission is 6 seconds instead of 30

Download Candidate Release WSJT-X 2.1.0-rc5  from http://physics.princeton.edu/pulsar/K1JT/wsjtx.html

Also, check out http://physics.princeton.edu/pulsar/K1JT/FT4_Protocol.pdf

I have already made several Qs on 20M within couple minutes.  It was like shooting fish in a bucket.

MSHV currently divides power output just as WSJT-X Fox/Hound does. 

If the Fox is transmitting 1KW, and he calls CQ, We all see his 1KW. if he drops to two streams, we all see 500 watts per stream. 
If he drops to four streams, we all see 250 watts per stream. 

It looks like the new Super Fox mode will be able to retain full power with multiple streams which would be a huge improvement.

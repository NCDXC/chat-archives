Last night, KQED showed the 1960 movie "Inherit the Wind". This
morning I read the wikipedia page describing that film, which led me
to the page describing the actual trial:
https://en.wikipedia.org/wiki/Scopes_Trial
where it mentions:
"Twenty-two telegraphers sent out 165,000 words per day"

165,000 (op x words / day) / (22 ops x 8 (hour/day) x 60 (min/hour))
is only 16 wpm !!! That seems low.

In the movie, there is a line of reporters on candlestick telephones,
instead of bugs. I suspect that was easier for the prop department,
and made for a more understandable scene.

As one who owns a TON of Mac gear, including a newly upgraded Macbook Pro, I share the disdain many seem to have for Windoze (aka the BSOD Bloatware Operating System).  It kills me that I can't run all of my ham software on a Mac Platform - in particular my K3 with LP-Pan & CW skimmer.  

Up to this point, I have been unable to make it fly, even running under different Virtual Machine systems. It crashes on launch with any machine that has to utilize a USB>Serial adaptor (and I have tried every one out there). Finally broke down and bought a stand-alone Dell w/ Windoze 7 to run most of my ham software.

I am hoping that my new MBP will finally allow me to migrate everything to my Mac.  There are a few serial port systems that will plug directly into the new ultra-high-speed "Thunderbolt" port on the Macbook.  This SHOULD allow me to run LP-PAN, Skimmer, HRD and other programs through a Virtual Machine on OSX. 

I've also heard about a new shareware utility called "Winebottler" that takes Windoze programs and converts them into something that functions just like a Mac App - that is, double click on an icon and the program launches without having to boot Widoze OS.  Just downloaded that one, and will let you know how it goes.

The MAC option is certainly viable, but if you don't want to buy expensive new hardware right now, you might try out (gasp!) Linux.   My XYL has all Apple stuff, and I keep a Windows box alive to run my Flex-3000, but everything else I've been very happy doing on Ubuntu Linux.   Been almost 20 years now since I defenestrated.

You can try it out on your current hardware, without disturbing Windows at all, by putting Ubuntu on a CD or USB memory stick, and booting your PC from there.  Tons of online support.   If you want to know how to <do x> just Google "ubuntu <do x>".

It will be somewhat slow running off the USB/CD memory, but you can see how it feels to use it.  If you like it, you can either install it over Windows and never look back, or install it beside Windows and have your choice of OS every time you boot up your machine.

So, as the saying goes ... Free.   No obligation.  No need to buy new hardware.   No time limit, just download and try it out. 

See http://www.ubuntu.com/download/desktop/try-ubuntu-before-you-install if you're interested or curious.

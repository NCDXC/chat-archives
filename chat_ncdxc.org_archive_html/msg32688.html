A little more info would give a better answer...

Crank up tower/mast?  Tilt over too?  Is there a utility box at the base with underground to the house/shack (for static protection too) so the run is short (BEST) or all the way to the shack (less than best)?

The point asked, does it need to flex?  (Flex is a relative term here.)

If so, low loss coax could be LMR 400 (it flexes for raising/lowering/tilting).  If you have a better budget, LD4 superflex, a half inch hardline (almost zero loss on a tower to a shack).  Note that each has specific connector needs, not generic (and some of THEM are not cheap).

Fixed tower?  There are other options.

ALL are budget choices but at the same time, scrimping on anything antenna, costs you too.  Which also defines 'best' overall; which compromises do you have to make?

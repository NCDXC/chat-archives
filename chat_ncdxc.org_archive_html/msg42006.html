Yes Gene.  This is a common problem with letters to USA.  
The problem is with the USPS sorting machine.  
Sometimes it is unable to read the address format.  
It sticks a yellow sticker on with garbage saying it cannot forward. 

Ideally this mail that gets rejected should be hand sorted for correct routing but USPS just return it.  
If you remember way back we formatted OQRS to correctly address USA letters to avoid this problem but I’d say 1 out of 100 get returned to me still.  
It needs to be highlighted to someone with influence at USPS.  

Note:
Tim is referring to the OQRS software developed by: PDXG, Tim M0URX and Charles M0OXO.
The software automates the QSL manger's back end processes, "The Bespoke OQRS", used exclusively by Tim and Charles.
The project began in 2013 by Pista HA5AO. Tim and Charles were enlisted for their QSL management expertise.
Over the years the system has worked flawlessly.

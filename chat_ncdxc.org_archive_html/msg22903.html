Hello all, it’s Truman N6TRU. This email is a repeat of a prior email for some of the people here, but will probably be new for others. I've started a new thread here so that my friend Garrett can respond if necessary. Anyways, he is working on a Sony BVH-500 NTSC video reel to reel system. I’ll repeat that any information is appreciated. The following message is directly from Garrett:

“Hello, I have been struggling to repair my BVH-500. My biggest problem is in diagnosing what is actually wrong with it. I get the error code ‘d7-7’ (which is to my knowledge a general error code) the drum spins up and the solenoid for the reel brakes engage and disengage (a few times) however the transport does not move. When I got it, I found that it had water damage (rusted ICs on the servo board). I replaced the broken chips, however that did not fix the problem. I have been unable to find any documentation on the BVH-500, after searching for a while through Sam’s and Google I can not find very much.”


Lastly, Garrett also wanted me to mention that he is willing to send any photos that might be necessary.


Thanks, and please inform if you have any info,

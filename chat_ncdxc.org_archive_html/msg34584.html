What operating systems (client and remote host)? I’m generally very happy with Microsoft Remote Desktop (RDP protocol, mstsc client). They’ve put a lot of effort into optimizing it over the years and it keeps up with all the advances in Windows – multi-monitor, multiple different dpis, etc. There are some tricks to get it to do more advanced things (e.g. registry edits to be able redirect USB devices), but it’s generally solid. It also has the advantage of being built into Windows, so nothing to download or install.

 

For remote audio though, especially with a laggy connection, you’ll want something UDP-based instead of TCP, so I use Sonobus for audio, with RDP for everything else.

> You don't need a two channel scope.  You can measure relative 
> phase and magnitude using the "3 voltmeter method".  You actually 
> only need one voltmeter, but it must have an RF probe, and it 
> has to be battery operated.  If necessary, 
> you can homebrew a diode detector to give any meter RF 
> capability.  Excite the array with a few watts of RF 
> and then measure the RF voltage between node A and ground, 
> then between node B and ground, and then between node A 
> and B.  The latter measurement requires the voltmeter to 
> be a floating battery operated version.  Alternately, 
> a differential RF probe can be used with any voltmeter. 
> Again, this can be homebrewed. 
>
> Given these three measurements, it is easy to use vector 
> algebra to calculate the relative amplitude and phase 
> of node A vs node B. 
>
> If you do want to use a scope, you should get one of 
> the modern low cost Chinese scopes that have a built 
> in phase measurement.  Not an old Tek 465, etc. 
>
> > Or maybe look into parasitic configurations instead? 
>
> Parasitic seems simpler (and it is) but will tend to have 
> less bandwidth and less front to back ratio.

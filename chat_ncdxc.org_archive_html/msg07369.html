Jim and everyone using DXKeeper,

I learned of the application to download from LoTW to DXK changing those
qualified from C to W.  Iain N6ML (who has a wealth of information and
readily shares it) told me about it back in 2011.  It works but obviously
can be.  It pretends to be a browser and looks for matches to the criteria.
As you can imagine it is fragile.  Any change to the webpage will cause the
application to fail.  So far - so good.

I have used it 2-3 times.  But I only use it if things look a bit out of
control otherwise I perform the task of getting Club Log from C to V as I
described.  Of course it makes my logbook accurate too.

Here is a clip from Iain's email with the link:

"Unfortunately DXKeeper currently has no way to extract the results of a
DXCC submission back out of LotW to update your log (changing the status to
'V'), but VE3NEA developed a hack to do this, called DXCC Credit Importer,
which you can find here:

http://www.dxatlas.com/Misc/DxCred.zip "

There is hope Dave AA6YQ can embed such a function in DXLabs.  

73,
Bob - W6OPO

It works the same for me, Bob. But there IS a semi-automatic process within
DXKeeper that downloads your current status from the ARRL DX desk, matches
new approvals with QSOs, and lets you approve the matches. 
I discovered this reading the help file. The time to do this is after you
have received you paper report from ARRL after a submittal.

Because I started over after moving here 8 years ago, and also because I
pursue band-countries, I've submitted a lot of QSOs each year, so the
semi-automatic process is easier. For OTs like you who probably submit only
a few, the manual process may be better.

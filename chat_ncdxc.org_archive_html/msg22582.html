The short answer for all DX contacts is, when in doubt, log them. This is particularly important for FT8, because there is some level of controversy about when to log the QSO, with some stations refusing to log until they have received a final 73.
This conundrum is an example of the "Two Generals' Problem" <https://en.wikipedia.org/wiki/Two_Generals%27_Problem>. The TLDR is that you can never be sure by waiting for the final message because that message will need an acknowledgement.
In contesting, where both ends of the QSO must either log or not log otherwise one party loses points from a NIL penalty, the problem is acute. Life in DXing is easier because there in no serious penalty for logging. The absolute worst is you send a paper QSL with 3 green stamps and received a NIL response many months later.
So, I say, when DXing or just general operating, let LotW or Clublog sort it out later.
Note that I always log when I send or receive a RR73 message. If you send RRR or 73 I will log.

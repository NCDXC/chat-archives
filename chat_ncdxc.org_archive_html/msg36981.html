Greetings -- we're going to schedule another test session for a WSJT-X RC7 pre-release this week, on Wednesday, 09/25, 1830 Pacific Time. Last week many stations successfully received the Fox transmissions, and were able to use the new verification scheme. With the help of participating stations, a number of issues were identified, and the WSJT-X development team has been working to resolve them. There's a new version available, and we'd like to try it out on Wednesday evening.
 
If you participated last week, we'd appreciate you joining again. If you didn't join us last week, please consider joining us this week. We've created a Groups.io group for testing pre-releases at https://groups.io/g/WSJTX-PreRelease-Testing - We'll use that venue to post pointers to new version of the pre-release software, collect error reports, .wav files, log files, etc.  (there's a message detailing this week's test, links to the software, what was changed, in that group already).
 
Please forward this message as appropriate, and please post questions about the pre-release test in the group, or to me directly: brian.moran@gmail.com
 
Thanks! 73

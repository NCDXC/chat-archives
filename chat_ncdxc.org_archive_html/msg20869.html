I note the cognitive dissonance of the word "simple" in the
same paragraph with "33 pages."  Browsing through it, my take
is that there is way too much unnecessary detail and the numerous
green text boxes clutter up the document.  ("More is less").
It is still probably better than ARRL docs. I am disappointed that there is no explanation of the various non-intuitive file suffixes LOTW uses. There is one that I can't remember that you have to know to save as a backup in case your LOTW files are corrupted. Otherwise, you have to start all over again. This document suggests that the suffix .tbk 
is the right one.  Maybe it changed, but it was definitely something
else originally, such as .tq6.

If you are still not on LOTW, set aside a LARGE block of time to
digest all this and act on it.

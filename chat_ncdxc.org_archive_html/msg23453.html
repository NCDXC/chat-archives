I’ll throw in one more vote for HRD logbook, although I do use more than just the logbook and pay for the current version because I like some of the newer features. It interfaces with WSJT to automatically log FT8 contacts, and handles auto-upload to LoTW and Clublog as well. And it makes it pretty easy to track DXCC and other awards. 

One option is to pay for HRD to get the most recent version and then not pay for the annual maintenance. That would give you all the latest features and any future bug fixes. You wouldn’t get new features, but could pay the maintenance fee at some point in the future if you decided you wanted something your version didn’t have.

Keith, I'm pretty much in agreement with you. I use Win4Yaesu for rig control of my FTDX10. It has a really nice GUI which is well thought out. It interfaces with the HRD logbook module, and I can even run DM780 if I want. I did upgrade to the latest HRD, though, as there are some changes to the logbook that are worth having, which have added functionality, brought compliance to the latest ADIF, and have corrected some older issues.
For contest logging, I just use the N3FJP suite of contest loggers, for which I bought a lifetime license, and for a very reasonable cost, I might add. I found N1MM to be too heavy for me, as I'm not a frequent contester.

The reason that we consolidate bookings via one path is to ensure that we get "credit" for all room-nights booked in participating hotels.  Those credits offset the rather significant costs of renting the Convention Center.  If you book outside of the system, it doesn't help us.

We're finalizing contracts for hotels now as well as updating to the NCDXC-version of the IDXC website.

We should be ready to go online Monday, June 16, using  the online Passkey reservation system, just like we did successfully in 2013.

On Mon, Jun 8, 2015 at 3:39 PM, larry moore via Chat <chat@ncdxc.org> wrote:
I think the rules for the DX program could well be improved.  Like:
1] The existing "non-country countries" should likely be re-designated so that the program would be a more equal distance/propagation problem no matter where you lived/operated from in the world. This would mean that large land masses like the USA, Canada, USSR, Australia, Brazil, China,Alaska etc would have several DX designations--- and many would go in areas like the Caribrean..  As a compromise [since it would not really change the overall challenge] You could keep all the existing "country concentrated hot spots" for history's sake -- ie Europe, Caribrean etc.
The vast landless Oceans would make it difficult to make the "world even" new countries --but it could be approxiated.
2] And the only contacts that should count would be only those made from one of those DX locations. This would mean, as an example, --- W1 & W6 contacts could not be pooled---same with VK2  &  VK6's. Everybody even.
3] It should be accepted IMO that no one--under these rules --would likely ever work them all.

To Russ's and Wayne's view on Mt. Athos.    Lots of "unfairness" exists in the present program.  Athos has one Ham. He/they wont let anyone else operate there [right now]. BUT----Tens of thousands have a SV/A card . Will it change?  Sometime.

P5          N. Korea is even worse.  BUT--Thousands have valid contacts. AND It has absolutely no Hams & they wont let outsiders operate.  By this definition it should go before SV/A does since it has one Ham. Will it change?  Somtime. 
And there's many others that are not much better.

I think our basic hangup is that we believe we have a "right" to work them all in X years-----or else it should be changed so we can.  I don't think the program would benefit from a yearly "paring and adding session" by the AARL.
Any other thoughts?

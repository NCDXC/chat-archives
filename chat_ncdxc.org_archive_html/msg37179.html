TI9 - Cocos Island
TI2JJP, Jose, is currently on Cocos Island. Previously he was a co-owner of the Oceania vessel, which is usually used to take tourists and expeditions to TI9. A last minute offer was made to Jose to go to Cocos Island, which he took. However while trying to set up the FT8 program Jose has apparently set up the callsign as TI9/I2JJP and some how forgot the T.

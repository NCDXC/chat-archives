I'm another one who has been tinkering / working with computers for awhile.  My "first" was a PDP-8, followed by a PDP-11/75.  The first machine I ever owned was a Colombia "VP" - a dual disk drive 8088 "portable" computer that folded into a package the size of a large suitcase.  I think it weighed in at about 50# plus or minus.  I can still remember feeling pretty cool hauling it around my client's offices.  Nothing impresses like cutting-edge high tech!  I've used just about every type of platform, but once I discovered Mac, I never went back.

As far as a decent desktop that will handle Windows XP & 7, you might look at the Dell Optiplex 760.  I found mine used on eBay for a couple of hundred bucks.  It has about everything you would need to run ham software (2.3GHz quad core processor, 8GB RAM, 500GB HDD.)  It is a decent performer, and there are many now on the surplus market.

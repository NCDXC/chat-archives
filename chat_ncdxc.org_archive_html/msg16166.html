For contesting -- N1MM Logger +
For DXing -- DXLab suite (I feel it to be a bit over-cluttered with its many floating applications) but in the end is quite powerful and has superior support.

Logger32 was fine back in the day.. but I have not used it for years so dont know where its ended up.


There are new(er) players emerging in this space but it remains to be seen if they can make the mainstream.  There is a wintest clone with better modern device support called dxlog.net that I have been playing with and watching it develop.

I’m interested in which logging program most people use if there is such a thing.  Okay don’t laugh but despite me being an engineer in silicon valley, I still use a paper log and just moved from a straight key to electronic.  In years past I would transfer the paper logs into DX4WIN but that program is pretty much defunct now.  So I started looking into programs as I hear hams mention their use and there’s a large variety.  Some of the top ones I’ve noted:

1.       ACLog

2.       TRX-Manager

3.       HamRadioDeluxe

4.       DXLab

5.       Logger32

6.       N1MMlogger

 

I’m sure there are more but I listed these as having good reviews and appear to have recent support.  My basic wish list would include:

1.        Integration with radio control to do RTTY and FT-8 would be nice (only concerned about current radios)

2.       Seamless uploads to LOTW

3.       Integration with online callbooks to immediately identify

4.       Automatic collation for award applications (I’ve never applied for any awards)

5.       Intuitive GUI use (DX4WIN was not intuitive)

6.       Label creation with current label printers

7.       I don’t care about contesting (understand some are more suited for that);  I’m purely a DX person only

8.       It doesn’t have to be free

 

After some time I’ll summarize the results and send out if I get enough responses.  Thanks in advance.

 

73s, Paul, W6IBU

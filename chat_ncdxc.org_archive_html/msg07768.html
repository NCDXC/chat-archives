Repeating what I've said earlier one more time; put a IRLP node at the site ($3-400, one time hardware expense), connect that to a reflector channel (free).  It requires site Internet access, now VERY common (and allows for secure repeater/site control).

That same hardware & software combo can also allow EchoLink access, but not at the same time as IRLP (do that via the reflector).

Then the club can argue, er, discuss over members only access, limited access or wide open.  Plus listen only reflector mode is another option. 

One of the primary reasons we're licensed is to communicate.  A semi-functional repeater with crippled coverage makes communication highly unlikely.  DX spotting on a repeater reaches more folks faster than email. 

It could allow the DX to coordinate better too, like the Iran attempt earlier this year, rather than Skype only in the background.

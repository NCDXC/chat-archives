Oh I knew this thread would get started, welcome to the party Paul, hi hi.

 

But first, congrats Paul, glad to see you there.

 

Second, I think it should be very hard to remove DXCC Entities just because there's no current licensing possibility.  Entities can be fluid.  What doesn't exist today will likely change tomorrow.  Unless there is an actual political or geographical reason to change an entity, I don't think it should be changed.  Doing so also takes away from the effort a ham may have made to get an entity that is temporarily not available today too.  I've heard arguments to take away SV/A for example because Monk Apollo passed away.  Yet there's been activity in the last year after all.  I expect the same will be true for North Korea eventually.  I've heard arguments to delete Spratly but now there's going to be activity there too.

 

Third, I can't think of a better person than Paul to participate and represent us as he's been to many if not all these places already.

 

73, Paul, W6IBU

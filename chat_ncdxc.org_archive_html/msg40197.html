I believe that if the ARRL offers and promotes an award that the Award should be attainable. Currently it is not possible (in a lifetime) to attain Honor Roll for anyone starting the journey now or in the recent past. Change is required when a program becomes stale.

 

Gene K5GS proposal to have an inactive list makes the most sense. It only requires a rule change and does not delete any entities. However, the outcry from the old timers of “NO CHANGE” is very emotional.

 

Compromise proposal: #1 Honor Roll rules remain unchanged and in effect becomes the legacy Award. You need to “Work them All” to receive the #1 Honor Roll status. No changes to the current rules for #1 Honor Roll. The rule to count entities for Honor Roll be changed to “you must have a total confirmed entity count that places you among the numerical top ten DXCC entities currently active in the last twenty years on the DXCC List” The Honor Roll has never been a “Worked them All” award. Honor Roll has always been an award for diligently “working the greatest amount of them but not all”. Additionally, a new award “High Honor Roll” could be created for working all currently active entities in the last twenty years.

 

In summary you would have three award levels (#1 Honor Roll, High Honor Roll, and Honor Roll).

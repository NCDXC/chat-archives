A long time ago when I was young and foolish, I picked up a 4-1000A at a swap meet on the basis of its "mystique".  Turns out the reality was not so great.  I started to build a homebrew amplifier for it but discovered that if I tried to run it in grounded grid (with screen grounded as well) the gain was poor that way unless I ran it with 5,000V on the plate.  It was never designed for that.  Also, standard bandswitched output tanks were not designed for 5,000V  Running it with the filament grounded required a screen voltage supply, and a neutralization circuit.

Anyway, it never got completed.

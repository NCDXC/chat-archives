We do DX by sending the radio as far away as we can, and see if it can still call home.

 

I am part of an organization called the California Near Space Project, CNSP. We do balloon launches to research how to make them fly Higher, Farther, and Longer. We have held the world record in all three of these categories and still hold the Distance and Duration records for a Latex balloon. We have launched Latex weather balloons that have flown from San Jose, CA to the coast of Algeria and another from San Jose to Morocco.

 

This week we launched a new type of small Super Pressure balloon that carries a 15 gram payload that contains a GPS, Radio transmitter, imbedded computer, Battery and solar panels. As I write this the balloon is in the Gulf of Mexico headed for Florida. It’s progress can be tracked with the following web link.

 

http://aprs.fi/#!call=a%2FK6RPT-11&timerange=604800&tail=604800     

Balloon update.

 I have made an attempt to predict it flight path for the next few days. Only time will tell if this comes true. The software I used is not meant for this type of balloon flight so I had to feed it some misinformation to get it to function on this flight. The balloon was launched from Almaden Expy. and Route 85 but was in Mexico when I did this prediction.

If you have any questions please send email to me at CNSP@donferguson.net or AI6RE@ARRL.net.

Hi Guys,

I'm one of the VP6A remote ops.....having a blast operating the RIB from home. I'm currently on 10M FT8 as I type this email.  SSB has been difficult because of latency issues.  The team has focused on money bands 10, 15, 20 as they work through the technical challenges.  Much progress has been made this week, with some of the world's best remote operators pitching in to solve both CW and SSB latency problems, including testing Mumble as an alternative app.  Current remote ops include CT1EEB, CT1BOH, KL2A, AA7A,  W1VE, F6EXV, PY5EG, PB8DX, K6UFO, N2IC W1VE, DJ4MX, VE5MX, and N1DG (Not N2DG  : >) -- among others.    Martti Lane, OH2BH, will soon join us.  Not sure if you'll see 12/17 SSB before this weekend, but I'll keep you posted.  This whole VP6A DXpedition has been an exciting real-time living laboratory --- bodes well for the future,
